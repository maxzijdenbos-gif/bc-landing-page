# Stylelint

[Link to official documentation](https://stylelint.io/)

Stylelint is a tool that helps you avoid errors and enforce conventions in your
styles.

## How to use

Run the `npm run stylelint` command.

## Features

### Linting

Stylelint statically analyzes your code to quickly find styling related
problems.

### Formatting

Stylelint can validate formatting issues.

### Fixing

Stylelint can auto-fix formatting issues.

### Ignoring

Stylelint can be configured to ignore entire files and directories.

### Configuring

Stylelint can be configured to use different rules and plugins to best fit your
needs.
