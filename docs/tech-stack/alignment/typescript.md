# TypeScript

[Link to official documentation](https://www.typescriptlang.org/)

TypeScript is a superset of JavaScript that compiles to plain JavaScript. It
offers classes, modules, and interfaces to help you build robust components. The
TypeScript language specification has full details about the language.

## How to use

Run the `npm run type-check` command.

## Features

### Interfaces

TypeScript uses interfaces to visualize what data is available to use.

### Type Checking

TypeScript validates that the types of values are compatible with the types
expected by the rest of your program.
